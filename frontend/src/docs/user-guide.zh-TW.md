# Cortex 系統操作手冊

版本：隨 Cortex 系統版本更新  
適用對象：一般使用者、部門主管、知識管理者與系統管理員

## 1. 登入與導覽

1. 開啟 Cortex 後輸入帳號與密碼。
2. 登入成功後，左側第一欄是主分類，第二欄是該分類的功能選單。
3. 系統使用 Hash URL；重新整理或直接開啟書籤時會保留目前功能位置。
4. 權限不足時系統會顯示錯誤，不會任意導向其他功能。

## 2. 系統作業 Tabs

- 每次開啟功能，畫面上方的「系統作業」會新增工作 Tab。
- 點選 Tab 可快速切換工作。
- 點選 Tab 右側的 × 可關閉工作。
- 在 Tab 上按滑鼠右鍵，可選擇「關閉此工作」或「關閉其他工作」。
- Tabs 依登入使用者保存在瀏覽器，重新整理後仍會恢復。

## 3. 上方快速功能

### AI Prompt

1. 點選上方 AI Prompt 圖示。
2. 輸入要交給 Codex 執行的工作指示。
3. 按「執行」後，畫面會即時顯示連線、執行步驟、標準輸出、錯誤輸出與完成狀態。
4. 執行中可按「取消工作」。
5. 此功能會在系統工作目錄內執行，僅管理員可使用。

### 操作手冊

- 點選 Help 圖示開啟本手冊。
- 可在繁體中文與 English 間切換。

### 即時通訊

- 點選即時通訊圖示會開啟「團隊協作 → 頻道與訊息」。
- 可切換工作區與頻道、傳送訊息、回覆討論串、加入表情反應及搜尋訊息。

## 4. 一般工作與儀表板

### AI 文件查詢

1. 輸入問題或分析要求。
2. 從左側文件樹選取一份或多份文件。
3. 執行查詢後，系統會進行相關文件檢索與跨文件整合。
4. 點選文件可預覽 Markdown、PDF、Word、Excel 或 PowerPoint 內容。

### 系統總覽、健康與活動

- 「總覽概況」顯示文件、索引、檢索、協作與圖譜統計。
- 「系統健康」顯示資料庫、Qdrant、模型服務及文件處理狀態。
- 「最近活動」彙整文件、檢索、研究、Issue 與協作活動。

## 5. 部門專區

系統內建 CEO、CFO、CTO、SALES、行政、人事、採購、MIS、業務工作專案、資訊專案及資訊安全專區。

- 每個專區有獨立 URL、統計與工作類型。
- 可新增、編輯、刪除、搜尋及依狀態或類型篩選工作項目。
- 建立者與管理員可修改或刪除項目。
- CFO、SALES、採購與部分專案功能可記錄工作金額。

## 6. 個人化專區

- 「我關注的文件」管理個人收藏與追蹤內容。
- 「等我審核」列出待目前使用者處理的審核工作。
- 「貢獻與積分」顯示個人知識貢獻紀錄。

## 7. 智慧對話與團隊協作

### 智慧對話

- 建立新對話後輸入問題，系統會使用知識庫內容回答。
- 對話歷史可用於回到先前工作脈絡。

### 頻道與訊息

- 建立工作區與公開／私人頻道。
- 管理頻道成員、即時訊息、討論串與表情反應。
- WebSocket 斷線時請確認後端服務與網路狀態。

### Issue 工作追蹤

- 建立 Issue，設定狀態、優先級、負責人與期限。
- 可加入留言、查看歷史與從「指派給我的 Issue」追蹤個人工作。

## 8. 文件與內容管理

### 工作目錄

- 在「根目錄」旁使用新增按鈕建立目錄。
- 目錄右側可重新整理、複製或刪除；刪除前系統會要求確認。
- 點選目錄後，該目錄成為目前上傳與操作目錄。

### 上傳文件

1. 選擇工作目錄。
2. 上傳 Markdown、PDF、DOCX、XLSX 或 PPTX。
3. Office 與 PDF 文件會先轉成 Markdown，再執行 PageIndex、分塊、Embedding 與向量索引。
4. 畫面會即時顯示 gRPC-over-WebSocket 處理階段與結果。

### 內容管理

- 可建立 Markdown、網頁或資料庫 SQL 內容。
- SQL 結果會轉換成 Markdown 後建立索引。
- 每次修改會保留版本，並可設定 RAG 與 PageIndex。
- 文件列表右側的刪除圖示可刪除文件。

## 9. 檢索、知識中心與圖譜

- 「全文檢索」使用文字條件搜尋。
- 「混合搜尋」結合關鍵字與向量語意。
- 「知識中心」管理文件、分類、草稿、審核、FAQ、專家黃頁及社群評價。
- 「知識圖譜」顯示文件與概念關係；社群分析與孤立節點可協助改善知識連結。

## 10. 深層研究與 AI 模型

- 深層研究可建立研究主題、執行多步搜尋並保存研究歷史。
- AI 模型功能可測試 Embedding 與 Reranking Endpoint。
- 索引工具可即時執行 GitNexus 或 Graphify 並查看輸出。

## 11. 系統設定

### 語言與主題

- 右上角可切換淺色／深色主題及繁中／英文介面。

### 系統參數

- 管理員可維護 Embedding、Reranking、PageIndex 模型及 API Base URL。
- API Key 僅能由後端環境變數或秘密管理服務設定，不會顯示於前端。
- 可維護畫面底部的聯絡窗口與常用連結，連結只接受 HTTP／HTTPS。
- 模型設定儲存後需重新啟動後端；底部資訊會立即更新。

## 12. 畫面底部資訊

底部列顯示版權、系統版本、聯絡窗口與常用連結。點選 Email、電話或連結可使用對應應用程式開啟。

## 13. 常見問題

- **登入後仍回登入頁：** 清除失效登入狀態後重新登入，並確認後端 JWT 設定一致。
- **文件一直顯示處理中：** 查看「系統健康」與上傳事件；確認 Embedding、PageIndex 與 Qdrant 服務。
- **無法使用 AI Prompt：** 此功能僅限管理員；確認後端可執行 `codex` 且已完成認證。
- **即時訊息中斷：** 確認 WebSocket 代理、後端連線與登入 Token。
- **設定無法儲存：** 僅管理員可修改；模型 Endpoint 與常用連結必須使用 HTTP／HTTPS。

## 14. 安全注意事項

- 不要把密碼、API Key、Token 或敏感個資貼入訊息、Prompt 或文件。
- AI Prompt 可操作系統工作目錄，執行前應確認指示範圍與預期變更。
- 刪除文件、目錄、工作項目或 Issue 前，請確認目標與影響範圍。
- 私人頻道與部門資料仍應依企業權限政策使用。
